# Power BI Sales Analytics Project

## Project Overview
Sales Analytics dashboard built in Power BI
connected to Databricks as data source.

## Tools Used
- Power BI Desktop
- Power BI Service
- Databricks (data source)
- DAX measures

## Dashboards Built
1. Sales Dashboard
2. Customer Dashboard

## Key Insights
- Total Sales: $29.36M
- Total Customers: 18.48K
- Top Category: Bikes (96%)
- Peak Month: December 2013
